# AddSearch Search UI Library

## Development
To modify the Search UI library, clone this repository and execute following commands.
#### Install dependencies
```sh
npm install
```

#### Run tests
```sh
npm test
```

#### Build
```sh
npm run build
```

Built bundle is saved under the *dist/* folder
